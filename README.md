# Medical Insurance Cost Prediction

## Problem Statement
Predict medical insurance charges based on customer demographic and lifestyle data to support pricing and risk assessment.

## Dataset
Public medical insurance dataset from Kaggle.

## Features
- Age, Sex, BMI, Children
- Smoking status, Region

## Approach
- Data preprocessing and encoding
- Exploratory Data Analysis (EDA)
- Feature relevance analysis
- Regression modeling (Linear, Ridge, Random Forest)

## Key Insights
- Smoking status is the most influential factor
- BMI and age significantly impact insurance costs
- Random Forest captured non-linear relationships best
